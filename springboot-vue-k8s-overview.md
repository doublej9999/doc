# Spring Boot + Vue 上 K8s 文档导航

这套文档已拆分为两种部署路径：

1. **在线部署版**（可访问公网镜像/仓库）  
   - 文件：`springboot-vue-k8s-online.md`
   - 适用：常规云环境、CI/CD 完整可用

2. **离线部署版**（内网隔离环境）  
   - 文件：`springboot-vue-k8s-offline.md`
   - 适用：政企内网、等保/合规隔离环境

---

## 如何选择

- 如果你的 K8s 节点可以直接拉 Docker Hub / 私有云镜像仓库：用**在线部署版**。
- 如果你的环境无法出网，需要镜像预下载 + 私仓导入：用**离线部署版**。

---

## 推荐阅读顺序

1. 先看在线版了解标准部署结构
2. 再看离线版理解镜像与 Chart 离线流程
3. 最后结合你当前环境形成统一的 Helm/Kustomize 模板
